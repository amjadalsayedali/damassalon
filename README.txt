Damas Salon – One-Page Website (index.html)

So nutzt du die Datei:
1) Lade index.html herunter
2) Doppelklick (öffnet im Browser)

Online stellen (einfach):
- Netlify Drop, Vercel Static, GitHub Pages oder klassisches Webhosting.
- Falls du eine eigene Domain willst: DNS auf den Hosting-Anbieter zeigen.

Wichtig:
- Telefon / E-Mail / Rechtstexte sind teilweise Platzhalter – bitte anpassen.
- Die Galerie nutzt Stockbilder (Unsplash). Ersetze die Bild-URLs mit eigenen Fotos.
- Das Kontaktformular sendet ohne Backend nicht wirklich ab (es kopiert Text / öffnet Mailprogramm).
